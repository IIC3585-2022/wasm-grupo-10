Bienvenido a Web-Partitioner

Este problema trata de particionar un conjunto de números en tripletas de modo que cada una de ellas sume el mismo número

¿Qué es una partición?

Una partición de un entero positivo n es una forma de descomponer n como suma de enteros positivos.

Las particiones son disjuntas y la unión de ellas debe producir el set S original
